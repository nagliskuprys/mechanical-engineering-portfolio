# Jacquard Weaving Machine Reverse Engineering

## Overview

This project involved reverse engineering a jacquard weaving machine based on measurements of a physical assembly.

The goal was to recreate the machine in CAD, identify the relationships between components, and produce technical documentation without access to original design files.

## Project Information

- Team size: 2 students
- Software: SolidWorks
- Components modelled: approximately 600–800
- Source data: physical measurements of an existing machine

## Responsibilities

- Measurement of physical components
- Creation of 3D part models
- Assembly modelling
- Identification of fits and mechanical interfaces
- Preparation of engineering documentation

## Skills Demonstrated

- Reverse engineering
- Mechanical assembly modelling
- Technical drawing interpretation
- Tolerance and fit analysis
- Team collaboration
- Problem solving
